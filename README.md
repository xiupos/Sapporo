# Sapporo

Xiupoy typing rule for libskk

## Install

```bash
mkdir -p ~/.config/libskk/rules
git clone https://github.com/xiupos/Sapporo ~/.config/libskk/rules/Sapporo
```

## Rules

### Symbols

- `,.` series

  | in | out | | in | out |
  | :-: | :-: | :-: | :-: | :-: |
  | , | `、` | | . | `。` |
  | z, | `‥` | | z. | `…` |
  | x, | `,` | | x. | `.` |
  | c, | `，` | | c. | `．` |

- `:;!?` series

  | in | out | | in | out | | in | out | | in | out |
  | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: |
  | : | `：` | | ; | `；` | | ! | `！` | | ? | `？` |
  | x: | `:` | | x; | `;` | | x! or x1 | `!` | | x? or x/ | `?` |
